# 情感云 Demo

## 说明

本项目为回车情感云服务 Demo，使用 Python 语言开发。仅作为开发参考示例。
使用本项目需要具备一下条件：

1. 拥有情感云服务 AppKey 和 Secret（如没有可联系相关运营人员）

Demo 运行基本流程为（通信部分需要遵循协议）：

1. 连接情感云服务
2. 建立会话
3. 初始化基础服务
4. 订阅基础服务数据
5. 启动情感云服务
6. 订阅情感云服务数据
7. 发送原始数据
8. 接收服务数据
9. 获取报表（可选）
10. 结束情感服务
11. 关闭会话

## 使用

1. 安装依赖
2. 配置 APP_KEY、SECRET 环境变量
3. 运行 client.py
